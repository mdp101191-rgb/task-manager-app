# 🚀 Task Manager App

A full-stack task management application built using React, Node.js, Express, and MongoDB.

## 🔧 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas
- API Testing: Postman

## ✨ Features
- Create tasks
- View tasks
- Update tasks
- Delete tasks
- RESTful API integration
- MongoDB database connection
